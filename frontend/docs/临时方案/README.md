## 🚀 使用指南

### 步骤1: 查看卡片
访问页面顶部，查看四个更新的指标卡片

### 步骤2: 选择配色
找到"🎨 图表配色方案"选择器，选择喜欢的配色

所有图表（饼图、柱状图）自动更新为新配色

鼠标悬停在饼图上查看详细的数值和百分比信息

## 📊 技术栈

- **框架:** Vue 3 + TypeScript
- **图表库:** Chart.js + vue-chartjs
- **样式:** Tailwind CSS
- **HTTP:** Axios

---

## 🎯 设计原则

1. **用户至上** - 关注核心指标
2. **视觉优先** - 美观的配色
3. **信息完整** - 绝对值+百分比
4. **交互友好** - 实时预览
5. **专业可靠** - 多场景适用
