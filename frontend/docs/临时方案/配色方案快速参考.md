## 📊 更新的四个卡片

1. **📥 总输入** - 输入Token总数 + 占比
2. **📤 总输出** - 输出Token总数 + 占比
3. **🔢 请求数** - API调用总次数
4. **💰 总费用** - 总成本金额

## 🎯 使用步骤

1. 打开用量统计页面
2. 滚动到"区间统计详情"
3. 找到"🎨 图表配色方案"
4. 选择配色方案
5. 查看右侧预览
6. 图表自动更新
