# 模拟测试模式 - 快速启用指南

## 🚀 快速启用（3步完成）

### 1️⃣ 编辑 .env 文件

在项目根目录的 `.env` 文件中添加：

```bash
MOCK_ANALYSIS_MODE=true
```

### 2️⃣ 重启Web应用

```bash
# 停止当前运行的应用（Ctrl+C）

# 重新启动
python start_web.py
```

### 3️⃣ 提交分析任务

- 打开Web界面
- 提交任何股票分析任务
- 观察进度显示（每步2-10秒随机延迟）

## ✅ 验证是否启用

查看日志，应该看到：

```
🎭 [模拟模式] 检测到 MOCK_ANALYSIS_MODE=true，使用模拟分析
🎭 [模拟模式] 开始模拟分析 AAPL（不执行实际任务）
```

## ❌ 禁用模拟模式

在 `.env` 文件中将值改为 `false` 或删除该行：

```bash
MOCK_ANALYSIS_MODE=false
```

然后重启应用。

## 📋 测试检查清单

- [ ] 时间戳秒数不同（不是整分钟）
- [ ] 每个步骤显示步骤用时（2-10秒）
- [ ] 总用时正确递增
- [ ] 进度条正确更新
- [ ] 步骤状态正确显示
