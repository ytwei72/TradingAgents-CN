## 2.2 健康检查

**状态**: ✅ 已完成

#### 请求

```http
GET /api/health
```

#### 响应

```json
{
  "status": "healthy",
  "version": "0.1.0",
  "timestamp": 1701504000
}
```

#### 响应字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| status | string | 服务状态 (`healthy` / `unhealthy`) |
| version | string | API 版本号 |
| timestamp | integer | Unix 时间戳 |

#### 示例

```bash
curl http://localhost:8000/api/health
```
